Web Application

A web application is a functional software program hosted on a server that allows users to perform specific tasks.

Its primary role is interactivity; it processes inputs, executes logic such as calculations, data filtering and generates unique output for the user.

  • Key Examples: Gmail, Google Docs and Netflix.
  • Structure: Often represented by icons and displayed through a browser interface
