Hypertext Markup Language (HTML)

HTML is the standard markup language used to create the structure of web pages[cite: 147, 151]. Its role is to use "tags" to tell the web browser how to display elements like headings, paragraphs, links, and images[cite: 148, 149, 150].

Essentially, it is the skeleton of every page on the World Wide Web[cite: 153, 154].

Common HTML Tags

Tag Description
<html> The root element of an HTML page[cite: 155].
<body> Contains the visible page content[cite: 155].
<p> Defines a paragraph[cite: 155].
