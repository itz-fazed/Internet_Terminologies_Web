File Transfer Protocol (FTP)

FTP is a standard network protocol used specifically for the transfer of computer data between a client and a server on a computer network.

While HTTP is for viewing websites, FTP is optimized for "bulk" tasks like uploading website files to a server or downloading large data sets reliably.
