Web 2.0: The Social Web

Web 2.0 marked a shift toward user-generated content, usability, and interoperability. It is often called the "Read-Write" web because it facilitates collaboration and sharing[cite: 167, 174].

Common examples include social media platforms, blogs, and wikis that allow for real-time interaction between users across the globe[cite: 167].
