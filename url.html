Uniform Resource Locators (URL)

A URL is the specific web address of a resource on the internet.

It provides a standardized, unique naming convention so that web browsers and servers know how to locate, retrieve, and display internet resources.

Parts of a URL

  • Protocol: Identifies the set of rules used for data exchange, such as https://.
  • Subdomain: Comes before the domain name and indicates a specific subdivision of the main website.
  • Domain Name: The human-readable name of the server representing the specific "address".
  • Top-Level Domain (TLD): The suffix that categorizes the website, such as .com for commercial or .edu for education.
  • Path: The string of characters following the TLD that specifies the exact location of a directory or file on the server.
