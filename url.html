Uniform Resource Locators (URL)

A URL is the specific web address of a resource on the internet[cite: 58, 59]. It provides a standardized, unique naming convention so that web browsers and servers know how to locate, retrieve, and display internet resources[cite: 60, 61].

Parts of a URL

  • Protocol: Identifies the set of rules used for data exchange, such as https://[cite: 198, 203, 205].
  • Subdomain: Comes before the domain name and indicates a specific subdivision of the main website[cite: 200, 209, 211, 213, 214].
  • Domain Name: The human-readable name of the server representing the specific "address"[cite: 201, 215, 218, 219].
  • Top-Level Domain (TLD): The suffix that categorizes the website, such as .com for commercial or .edu for education[cite: 204, 220, 223, 228].
  • Path: The string of characters following the TLD that specifies the exact location of a directory or file on the server[cite: 199, 224, 225, 229, 230].
