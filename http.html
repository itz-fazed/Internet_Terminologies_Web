Hypertext Transfer Protocol (HTTP)

HTTP is the foundational application layer protocol for transmitting hypermedia documents, like HTML files, across the internet.

It establishes a set of rules that dictate how messages are formatted and sent between web browsers and web servers, ensuring both sides understand each other.

The Request-Response Cycle

  • Client Request: The browser sends a message (like an HTTP GET) to the server asking for a file.
  • Server Response: The web server processes the request and sends back a status code (like 200 OK for success or 404 Not Found if the page is missing) along with the requested data.
