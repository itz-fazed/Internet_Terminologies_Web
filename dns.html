Domain Name System (DNS)

DNS functions as the Internet's directory or phonebook.

It translates human-readable domain names, like www.chess.com, into machine-readable IP addresses like 192.0.2.1.

This allows computers to identify each other on a network.

Its primary role is to save users from having to memorize complex strings of numbers to visit websites.
